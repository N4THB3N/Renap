Etiqueta font

La etiqueta font es util para cambiar la apariencia de la letra dentro de nuestro documento podemos cambiar el tamano de la letra, el color, el tipo y demas cosas.

Propiedad Descripcion
family Este es un atributo que cambia el tipo de letra que esta en el documento.
size Este atributo cambia el tamano de la letra que nosotros estamos visualizando en el documento.
color Este atribu cambiar el color de la letra de un documento.
