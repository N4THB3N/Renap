Etiquetas h

Las etiquetas h sirven para anadir encabezados al documento. Estas etiquetas van desde la etiqueta h1, hasta la h6, cada una de estas etiquetas tienen distintos tamanos la h1 es la que tiene el tamano mas importante. h1 es un texto muy grande
h2 es un texto grande
h3 es un texto un poco mas grande de lo normal
h4 es un texto de tamano normal
h5 es un texto de tamano pequeno
h6 tiene un tamano muy pequeno

Propiedad Descripcion
Color Esta propiedad hace que el color de le letra cambie para todo el documento.
Background color Esta propiedad cambia el color de fondo de todo el documento.
Etiqueta center Esta etiqueta permite que el texto este centrado. Esta etiqueta necesita apertura y cierre.
