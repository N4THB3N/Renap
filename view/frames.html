Algunos frameworks, mismo fin

Cada una de estas herramientas permite entregarle al cliente un mejor resultado.

Bulma

Este es un framework basado en CSS, totalmente responsivo. Bulma se enfoca al movil lo que hace que cada elemento este optimizado para acomodarse de forma vertical sin usar clases extra.

La ventaja de este framework es que esta basado en FlexBox. Esto ayuda a objetener un diseno flexible con columnas del mismo tamano. Todo el diseno hecho a traves de Bulma tiene su base en clases.

Materialize

Este framework es capaz de realizar codigo mas consistente, y mejor. Permite a los disenadores crear aplicaciones con las guias de Material Design. Estas guias son propuestas de como deberia de ser la interfaz del usuario en los dispositivos android.

Esta herramienta es mas nueva que las demas, mas ya ha ganado nombre. Las guias de Materialize ayudan en gran manera al usuario a guiarse con base en el diseno de las paginas.

Foundation

La ventaja de este framework se ajusta de forma dinamica. Este framework tiene la gran dicha de ser el precursor de diseno, este anade las estructuras necesarias mas normales maqueteando un sitio rapidamente.

Es modular, lo que conlleva que sea una serie de hoja de estilos de CSS, estos conllevan consigo la implementacion de diferentes componentes de la caja de herramientas.

Ventajas y desventajas

  • Facilita el proceso de desarrollo de aplicaciones web.
  • Ayuda a reutilizar el codigo.
  • Permite abstraernos del navegador, para solo enfocarnos en el codigo.
  • Saber utilizar un framework no implica saber como funciona ni sus tecnologias.
  • La relevancia de los frameworks se sujeta a modas, pero puede decaer el siguiente ano.
