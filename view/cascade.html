CSS es una herramienta utilizada por muchos programadores y gente que se dedica al oficio del diseno en paginas web. Esto es utilizado para mejorar todas aquellas visualmente todas las carencias que puedan tener algunos objetos en HTML.

Hasta el momento han salido tres versiones de CSS la tercera pretende brindar soluciones a las carencias que se presentaban en modelos anteriores. Las mejoras mas notables pueden empezar desde brindar mejor comptabilidad con los navegadores web.

Degradados y Animaciones

Estos efectos que se lograron en estas imagenes, se llevaron a cabo con la combinacion de ciertas propiedades de CSS3 que nos permiten ver paso a paso los cambios que se realizan sobre un objeto.

El selector llamado hover, se utiliza para cuando el mouse es puesto encima de un elemento y al hacer esto, se empiezan a desencadenar las acciones predeterminadas por este selector. En este caso, se usaron las propiedades de transition con un tiempo, asi como tambien transform con un border radius y demas propiedades para los bordes.

Rotaciones y Efectos

Aqui se utilizaron etiquetas tipo span, y fueron manipuladas por las intrucciones de CSS. De igual forma que hace un rato, para hacer esto se utilizo una transicion determinada con un tiempo determinado.

Para esto tambien se utiilzan unos keyframes estos nos utilizan para poder pasar la transicion o dividirla en momentos, en esos momentos es que toma una breve pausa y luego realiza las demas funciones predeterminadas, a cada uno de estos segmentos de tiempo se les anade una rotacion a las imagen, para que de la impresion de que esta rodando.
