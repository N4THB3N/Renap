Etiqueta input

Esta etiqueta es utilizada para todos aquellos objetos de entrada, es decir todos aquellos objetos que reciben informacion. A esta etiqueta hay que definirle que tipo de objeto de entrada nosotros queremos que sea, ejemplo: un campo de texto, o un boton circular en el que el usuario puede escoger una opcion.
