Etiqueta select y optgroup

Etiqueta select

La etiqueta p hace referencia a un objeto contenedor. Este objeto despliega una lista al darle un clic sobre el mismo. En otros editores graficos es conocido como Combo Box.

Etiqueta optgroup

Esta etiqueta tambien va dentro de la etiqueta select y lo que hace es que agrupa una lista dentro de la lista que ya tenemos. Es decir permite crear sub grupos dentro de la lista.
