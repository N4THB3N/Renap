Etiqueta body

La etiqueta body es necesaria en cada documento html porque sin esta es imposible agregar elementos dentro del formulario. Esta etiqueta hace referencia a todo el cuerpo del documento, sin ella, no podemos agregar elementos.

Propiedad Descripcion
Color Esta propiedad hace que el color de le letra cambie para todo el documento.
Background color Esta propiedad cambia el color de fondo de todo el documento.
Background Esta propiedad nos permite colocar algun elemento de fondo en nuestro documento por ejemplo, se le puede anadir la direccion de una imagen.
