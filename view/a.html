Etiqueta A

Esta etiqueta sirve para dar un hiperenlace, es decir, re digirigirnos hacia otra pagina, esto se hace a traves de agregarle la referencia a la pagina a traves de href y el URL o el nombre de archivo que queremos abrir. Tambien es importante que veamos que dentro de las etiquetas de apertura y cierre podemos colocar cualquier texto para la referencia a la pagina no sea visible.

Propiedad Descripcion
target Este es un atributo de la etiqueta a, que busca una ventana o un frame que concida con el nombre que se le ha asignado.
hreflang Este atributo puede llegar a definir el lenguaje del documento con el que se esta enlazando.
