Etiqueta button

Esta etiqueta permite que se puedan agregar botones dentro de una pagina. Podemos agregar dentro de esta etiqueta varios atributos y texto para que este dentro del boton, tambien hay manera de cambiar el tamano del boton y demas configuraciones.

Para que podamos utilizar esta etiqueta debemos de tener en cuenta que se utiliza la etiqueta de apertura y de cierre.

Propiedad Descripcion
br Se le puede colocar a un boton un salto de linea, este afecta el tamano del boton.
font Este atributo permite que podamos modificar la letra que esta dentro de un boton
img Este atributo permite colocarle una imagen al boton, esto ayuda con el diseno de la pagina.
background color Esta etiqueta permite que le podamos anadir un color al boton
