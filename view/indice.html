SOLO HTML

HTML son las siglas de HyperText Markup Language. Es utilizado como una tendencia en casi todas las paginas que se visitan diariamente. No es un lenguaje de programacion como tal, sino un lenguaje de etiquetas.

HABLEMOS DE CSS

Las siglas CSS significan Cascade Style Sheets, HTML es un lenguaje que podemos aplicar para dar estructura, sin embargo, CSS nos ayuda a darle un diseno unico a lo que hicimos. Esta parte funciona a traves de modulos que pueden ser utilizados en cualquier necesidad.

UTILIZA FRAMEWORKS

Ahora queremos hablar un poco acerca de los Frameworks. Un Framework es un entorno o un espacio de trabajo. que tiene estandares, practicas y criterios para manejar cierto tema. Estos pueden ser usados en HTML y CSS la aplicacion de los mismos resulta muy util cuando se trata de estructuras complejas.

JAVASCRIPT

Es un lenguaje de programacion que es utilizado para crear paginas web, mas interactivas la utilizacion de este lenguaje de programacion permite dar una mayor actividad a la pagina con el cliente. JavaScript no debe de ser confundido con Java porque tiene sus diferencias por ejemplo, este no necesita de un compilador.
